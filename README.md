# Netflix Exploratory Data Analysis (EDA)

## Problem Statement
Analyze Netflix Movies and TV Shows dataset to identify trends and patterns.

## Dataset
Netflix Titles Dataset from Kaggle.

## Objectives
- Summary Statistics
- Correlation Analysis
- Trend Identification
- Data Visualization

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Google Colab

## Key Insights
- Netflix contains more Movies than TV Shows.
- Content growth increased significantly from 2016 to 2019.
- United States contributes the highest number of titles.
- TV-MA and TV-14 are the most common ratings.
- International Movies and Dramas are among the most popular genres.

## Conclusion
EDA helped identify major trends, content distribution, and viewing patterns within Netflix's catalog.
